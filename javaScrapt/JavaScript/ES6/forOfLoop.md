#### for.. of loop javascript/ES6

Here are all types of for loops in JS:

```javascript
// for
for (let i = 0; i < arr.length; i++) {
  // i is index, arr[i] is element
}


// for
..in
for (let i in arr) {

  // i is index, arr[i] is element

}


// for
..of
for (let elt of arr) {

  // index not available, elt is element

}


// forEach
arr.forEach((elt, i) => {

  // i is index, elt is element

});
```
